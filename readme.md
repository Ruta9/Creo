# Creo 
Personal self-learning project.

### Technologies used:
* Java
    - Spring
    - Spring Boot
    - JPA (Hibernate)
* JavaScript
    - React
    - Redux
    - Redux Thunk
* CSS
    - Semantic UI framework

Things to still try out:
- Unit testing

- Web Sockets + sockJs

A simple project management tool. A mix between Trello, Jira and ServiceNow.
